中→日 変換規則

語彙
    去X → はX
    且 → かつ
    其余 → それ以外
    >=N → N以上
    <=N → N以下
    =N → N (上位条件で N+1以上 が処理済前提)
    高速或以上舰队 → 高速以上の艦隊
    高速舰队 → 高速艦隊
    低速舰队 → 低速艦隊
    最速舰队 → 最速艦隊
    ?, ？ (不確実マーク) → オミット
    札についての情報は無視せよ

艦種略号
    src\models\Composition.ts を参照
    複合略号の例:
        SS+SSV / SS系 → 潜水系
        BB系 → 戦艦級
        CV系 → 空母系
        CA系 → 重巡級
        DD+DE → 駆逐+海防
        CL+CT → 軽巡+練巡
        CL系 → 軽巡+練巡+雷巡
        AV+AS → 水母+潜水母艦
        CV系+AV+CAV → 空母系+水母+航巡

確率表記 (2-2.txt を模範とする)
    単独宛先・確定: はX:N%
        例: 航戦1はB:70%
    単独宛先・概算 (原文の 约/約 に対応): はX(N%)
        例: 軽巡+練巡1以上 かつ 駆逐+海防4以上はE(60%)
    複数宛先: はX:N%,Y:M%[,Z:K%]
        例: 戦艦級+空母系3はG:70%,K:30%
        例: 駆逐+海防1はI:35%,J:15%,K:50%

書式
    インデント: タブ
    複数の子条件を持つ場合: 「〜の場合、」+ インデント
        例:
            戦艦級+空母系1の場合、
            駆逐+海防2以上はG:30%,K:70%
                駆逐+海防1以下はG:50%,K:50%
    更新日時: ファイル末尾に「最終更新: YYYY-MM-DD HH:MM」
